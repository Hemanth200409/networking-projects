# Network Connectivity Checker

## Description

This Python tool checks whether a host (website or IP address) is reachable using the ping command.

## Technologies Used

* Python
* OS Module

## How to Run

1. Run the Python script.
2. Enter a website or IP address.

Example:

Enter website or IP: google.com
Host is reachable

## Purpose

Helps verify network connectivity between a client and a server.
