# TCP Port Scanner

## Description

A Python tool that scans common TCP ports on a target host to identify open services.

## Technologies Used

* Python
* Socket Programming

## Ports Scanned

21 - FTP
22 - SSH
80 - HTTP
443 - HTTPS

## How to Run

1. Run the script.
2. Enter a website or IP address.
3. The program displays open and closed ports.

Example Output:

Scanning google.com
Port 80 is OPEN
Port 443 is OPEN
