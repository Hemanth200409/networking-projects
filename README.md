# Networking Projects

This repository contains Python-based networking tools developed to demonstrate basic network automation and troubleshooting concepts.

## Projects Included

### 1. Network Connectivity Checker

A Python tool that checks whether a host (website or IP address) is reachable using the ping command.

Features:

* Tests connectivity to a remote host
* Simple network troubleshooting tool
* Uses Python OS module

Location: `connectivity-checker/`

---

### 2. TCP Port Scanner

A Python script that scans common TCP ports on a target host to detect open services.

Features:

* Scans common ports (21, 22, 80, 443)
* Uses Python socket programming
* Helps identify running network services

Location: `tcp-port-scanner/`

---

### 3. Network Monitoring Tool

A Python monitoring script that continuously checks server availability.

Features:

* Periodically checks connectivity
* Displays server status (UP / DOWN)
* Simulates basic network monitoring systems

Location: `network-monitor/`

---

## Technologies Used

* Python
* Socket Programming
* OS Module
* Network Troubleshooting Concepts

---

## How to Run the Projects

Make sure Python is installed.

Example:

```
cd connectivity-checker
python connectivity_checker.py
```

Follow similar steps for other tools.

These projects were created to practice networking fundamentals and automation using Python.
